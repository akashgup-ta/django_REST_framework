I have created Django model with 10 different types of field ( char, int, float, file), create a rest api to list all object of model where particular field is null

test api with postman. And in postman output it will display list of fieldnames which are null

http://127.0.0.1:8000/app1/emp/



Methods


GET method - by using GET we get all the data, Example : http://127.0.0.1:8000/app1/emp/

To get particular data (only one record), change to get request by id we can get particular data | Example - get request by id : 127.0.0.1:8000/app1/emp/1/


POST method - by using POST we can add data | Example - 127.0.0.1:8000/app1/emp/ , after write json


PUT method - by using PUT we can update data (complete update) | Example - to update record 2 : 127.0.0.1:8000/app1/emp/2/, after write json


PATCH method - patch method is used for particial updation, it means we can update only one record of particular record | Example - to patch record 2 : 127.0.0.1:8000/app1/emp/2/, after write json


DELETE method - we can delete only one record at a time | Example - to delete record 3 : 127.0.0.1:8000/app1/emp/3/
